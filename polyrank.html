polyrank.mmatrix module¶

class polyrank.mmatrix.MMatrix(mat)[source]¶

Bases: builtins.object

Decorator class for a Sage matrix of type sage.structure.element.Matrix.

Provides immutable infix operations by returning a new MMatrix that represents the result. Specifically, the infix operations +, -, and * are immutable.

Matrix rows can be retrived and set using the [] notation, note that setting a row is not immutable.

Module contents¶

The main purpose of this package is to calculate the conditions under which a polynomial matrix has rank less than or equal to n.

The main entry point is main.find_conditions_rank_leq_n().
